установка зависимостей:
```
npm i
```

запуск проекта:
```
npm run serve
```

сборка проекта:
```
npm run build
```

сборка проекта с минификацией файлов:
```
npm run build:min
```

особенности сброки:
```
Webpack 5
Нативный JS: Babel
Разметка: Pug
Стили: Css / Sass / PostCSS
```

псевдонимы:
```
путь к папке images: ~img
путь к папке fonts: ~fonts
```
